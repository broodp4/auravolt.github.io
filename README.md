# AuraVolt — Open Source Solar Website (MIT)

A clean, responsive, multi-page static site for a solar installation business.

**Features**
- Modern, accessible, mobile-first design
- Plain HTML + CSS + tiny JS (no build tools)
- OpenSolar lead capture widgets integrated on the home & contact pages (and included globally)
- Ready for GitHub Pages, Netlify, or any static host

**Pages**
- `index.html` — Home/landing with lead widget surface
- `services.html` — Service offerings
- `about.html` — Company overview
- `faq.html` — Common questions
- `contact.html` — Native form + OpenSolar widget surface

## Deploy on GitHub Pages
1. Create a repository and push these files.
2. In GitHub: Settings → Pages → Source: GitHub Actions (or just serve as static files).

## Customize
- Update copy, pricing ranges, and service area.
- Replace `assets/img/logo.svg` with your brand.
- Point the native contact form to your Formspree endpoint (in `contact.html`).
